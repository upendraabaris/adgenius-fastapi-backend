# AdGenius FastAPI + MCP Project (User-specific MCP agents)

This project is a scaffold for a FastAPI backend that:
- Uses SQLite + SQLAlchemy (for simplicity)
- Implements JWT auth (login/signup)
- Stores business profile and integrations (Meta access token and ad account id)
- Demonstrates user-specific MCP agent usage using `mcp_use` and `langchain_google_genai`
- Provides endpoints: /auth, /business, /integrations, /chat

**Important:** This is a scaffold. Replace placeholders and install real packages.
See `.env.example` for required environment variables.
